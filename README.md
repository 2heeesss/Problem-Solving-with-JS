## 배열

### include() 메소드
배열 안에 값이 있는지 확인
값이 있다면 true, 값이 없다면 false를 반환한다.
```js
배열.include(값)
```
### indexOf() 메소드
전달받은 값과 동일한 배열 요소가 처음으로 등장하는 위치의 인덱스를 반환한다.
```js
배열.indexOf(값)
```
